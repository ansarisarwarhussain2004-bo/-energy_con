# 🔋 Appliance Energy Prediction

## 📌 Overview
Machine Learning project to predict appliance energy consumption using environmental data.

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Features Used
- Temperature
- Humidity
- Time-based data

## 🚀 How to Run
1. Open Jupyter Notebook
2. Run all cells

## 🖥️ Frontend Options
- Streamlit UI: `streamlit run app.py`
- Flask web UI: `python app_flask.py`

## 🎯 Goal
Predict energy usage of household appliances based on conditions.
